# 🚁 Стенд тестирования квадрокоптера

Система мониторинга параметров квадрокоптера на испытательном стенде в реальном времени.

## 👥 Авторы
- **Полковников А.В.**
- **Глазырин К.С.**

**Организация:** Кафедра робототехники и противодействия робототехническим комплексам (системам)

---

## 📊 Измеряемые параметры

- 🔄 **Обороты двигателей** (4 датчика, RPM)
- 🌡️ **Температура двигателей** (4 термистора, °C)
- ⚖️ **Тяга** (2 тензодатчика, граммы)
- 📊 **Вибрация** (акселерометр 3 оси, м/с²)
- 🔊 **Уровень шума** (микрофон, дБ)

## 🚀 Быстрый старт

### Вариант 1: Прямое подключение к Arduino (рекомендуется)

1. **Настройте Arduino:**
   ```bash
   # Откройте arduino/quadcopter_stand.ino
   # Измените IP адрес в config/config.json
   # Загрузите на Arduino
   ```

2. **Настройте конфигурацию:**
   ```json
   // config/config.json
   {
     "connection": {
       "mode": "direct",
       "direct": {
         "arduino_ip": "192.168.1.100"
       }
     }
   }
   ```

3. **Запустите веб-интерфейс:**
   ```bash
   # Откройте frontend/index.html в браузере
   # или используйте локальный сервер:
   python -m http.server 8000
   # Откройте: http://localhost:8000/frontend/
   ```

### Вариант 2: Через Python сервер

1. **Установите зависимости:**
   ```bash
   pip install pyserial matplotlib numpy netifaces
   ```

2. **Настройте конфигурацию:**
   ```json
   // config/config.json
   {
     "connection": {
       "mode": "server"
     }
   }
   ```

3. **Запустите Python сервер:**
   ```bash
   cd backend
   python monitor_with_web.py
   ```

4. **Откройте веб-интерфейс:**
   ```
   http://localhost:8000/frontend/
   ```

- Arduino Nano
- Ethernet модуль W5500

- Arduino IDE 1.8+
- Библиотеки Arduino:
  - Ethernet
  - ArduinoJson
  - HX711
  - Adafruit_ADXL345
- Python 3.7+ (для серверного варианта)
- Современный браузер (Chrome, Firefox, Edge)

## 🐛 Отладка

### Проблема: "Нет связи"
1. Проверьте IP адрес в `config/config.json`
2. Убедитесь что Arduino в той же сети
3. Проверьте endpoint: `http://ARDUINO_IP/data`
4. Откройте консоль браузера (F12)

### Проблема: Неверные данные
1. Проверьте калибровку в `config/config.json`
2. Проверьте подключение датчиков
3. Проверьте Serial Monitor Arduino

Подробнее: [docs/SETUP_GUIDE.md](docs/SETUP_GUIDE.md)
